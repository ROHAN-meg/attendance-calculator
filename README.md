**Attendance Calculator**
A clean and easy-to-use web app built to help college students quickly track and understand their attendance without any hassle.

** Features**
Calculates attendance instantly with accurate results
Works for both theory classes and lab sessions
No browser extensions or extra setup needed
Responsive design that works smoothly on all devices

** Tech Stack**
HTML
CSS
JavaScript
